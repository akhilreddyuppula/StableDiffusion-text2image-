# StableDiffusion-text2image-



Hi,
this is a super super extra mini small prac project on a basic diffusion model using stable diffusion to generate images from text prompts.
For the test images i've prompted the model to generate images of a dog and cat dressed up as wizards which can be seen in the attached images in this rep.

Modules used:
diffusers
torch
